企业级路由设备

地铁？2号线 厚庭 周日

IP地址 作用？在网络中 唯一标识一台设备 接口属性 一台企业级路由器可以有多个接口
32位二进制数组成 点分十进制

网段？ 将IP地址和子网掩码 进行与运算
192.168.1.1 255.255.255.0 - - > 192.168.1.0
172.16.2.2 255.255.0.0 - - > 172.16.0.0
10.1.4.6 255.0.0.0 - - > 10.0.0.0

子网掩码表示形式
255.0.0.0 = /8
255.255.0.0 = /16
255.255.255.0 = /24

企业级路由 作用：分割广播域
什么叫广播域？ 1个网段 = 1个广播域
路由器的一个接口 代表一个广播域
默认情况下，路由器只能和同一网段的设备相互通信

路由器通信原理？？？
路由表 - 相当于我们现实生活当中的地图
Router#show ip route - 查看路由表

路由表当中记载的就是本台设备能够访问的目的网段的信息

路由来源 - 路由来源指的是通过什么方式 学习到的该信息
C 直连
S 静态路由

一、静态路由技术 指路 实现了企业级路由器设备的跨网段通信
ip route 目的网段 目的网段的掩码 下一跳
Router(config)#ip route 192.168.2.0 255.255.255.0 192.168.1.2

下一跳 ： 下一跳指的是 正确方向上的对端路由器接口的IP地址

网络通信 有去有回的过程

二、DHCP
Dynamic Host Configuration Protocol 动态主机配置协议
作用：给终端设备自动下发上网参数 可以实现局域网通信

上网参数获取方式
①早期 静态手动配置
②动态自动获取

校园网？
如果说校园网认证没有通过
①能不能上网？
②能不能访问校内的服务器？

家用路由器发射出来的无线信号
②城域网 MAN 50km
行程码

③广域网 WAN 最大的网络

上网参数
按下键盘上的windows+R 弹出来一个运行 输入cmd ipconfig/all
1.IP地址
2.子网掩码
3.网关 针对终端设备 本质上是一个IP地址 在终端用户进行跨网段通信的时候至关重要
网关是一个广播域的边界
4.DNS 域名解析服务器
作用：将域名 解析成IP地址 供用户访问
www.badu.com

218.85.157.99 福州电信
8.8.8.8 谷歌
114.114.114.114 早期公共

DHCP 地址池 pool 一个地址池 只能给一个网段用户去使用
Router(config)#ip dhcp pool ND - - 创建一个地址池，名字叫ND
Router(dhcp-config)#network 192.168.1.0 255.255.255.0 –配置下发的IP地址所处的网段和子网掩码
Router(dhcp-config)#default-router 192.168.1.100 – 配置下发的网关参数信息
Router(dhcp-config)#dns-server 8.8.8.8 - - 配置下发的DNS参数信息
Router(dhcp-config)#exit

出现以169开头的地址，代表这台设备想通过DHCP的方式，动态获取上网参数，但是获取失败

show run – 查看当前设备配置的所有信息
